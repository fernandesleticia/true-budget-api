# True Budget API 
Provides API for [True Budget App](https://github.com/hhldiniz/true-budget)

## Prerequisites
- Ruby 2.7.4
- Rails 6.1.6
- PostgreSQL 12

## Create env files
```
cp "sample.env" ".env"
```

To generate your DEVISE_JWT_SECRET_KEY:
```
rake secret
```
